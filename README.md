# DNS Stamp

Ruby implementation of [DNS Stamp](https://dnscrypt.info/stamps-specifications)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'dns_stamp'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dns_stamp

## Compatibility

DNS Stamp support ruby 2.6+
